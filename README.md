# Binaryrs

Trading bot for Binary.com done in Rust

## Run

```bash
cargo run
```

## Usage

Need to have ENV variables setup in your environment, your binary.com `token` and `app_id`

```bash
export BINARY_TOKEN=XXXXXXXX
export BINARY_APP=XXXXX
```
